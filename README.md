# PEX CARD PHP SDK
This is a PEX CARD PHP SDK to trigger the api



[View tutorial](https://developer.pexcard.com/docs4)

## Usage
- Install package

```
composer require aatishgore/PexCardSDK
```
- Publish Vendor

```
php artisan vendor:publish
```
